# noteバナー生成 Codexスキル 簡易版 使い方

## これは何？

noteのプロフィールヘッダー画像を、公式推奨サイズ `1280×670px` で作るためのCodex用スキルです。

画像生成だけではサイズや文字位置がズレることがあるため、このスキルでは最後にCodex側で完成画像を整えます。

## できること

- noteプロフィール用バナーを作る
- 完成PNGを `1280×670px` で出す
- 中央の `1280×216px` に重要な文字や人物を収める
- 1枚または複数枚を指定して作る
- テイスト指定にも対応

## 使い方

1. `note-banner-maker-skill.zip` をダウンロードします。
2. ZIPを解凍します。
3. 解凍して出てきた `note-banner-maker` フォルダを、Codexのスキルフォルダに入れます。
4. Codexを再起動、または新しいチャットを開きます。
5. 下記のように依頼します。

```text
noteプロフィール用バナーを作ってください。

枚数：1枚
方針：おまかせ
サイズ：1280×670px
重要な文字や人物は中央1280×216pxに収めてください。
```

## 最初に聞かれること

全部答えなくても大丈夫です。

```text
・アカウント名：
・肩書き：
・ターゲット：
・実績・強み：
・一番伝えたい価値：
・雰囲気・色味：
・人物・モチーフ：
・参考画像：
```

## 依頼例

```text
おまかせで1枚作る
```

```text
おまかせで複数枚作る。枚数は3枚でお願いします。
```

```text
テイスト指定で1枚作る。上品で清潔感のある白×淡いブルーのボイトレ系バナーにしてください。
```

## 注意

- 完成画像だけをnoteにアップロードしてください。
- ガイド線入り画像は作らない設計です。
- 画像生成を使う場合、利用環境によっては生成回数や利用料が発生する場合があります。
