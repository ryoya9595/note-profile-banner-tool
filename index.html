Codex Skill / note Header

「noteで見切れがちなプロフィールバナー問題」を解決！

Codexにこのスキルを入れて起動するだけ。「どんなバナーにする？」「何枚作る？」と Codexの方から聞いてくれるので、答えていくだけで、そのままnoteに設定できる完成画像 （1280×670px・文字が見切れない設計）ができあがります。

Step 0

まずはCodexアプリを入れよう

「Codex（コーデックス）」は、あなたのパソコンの中で作業してくれるOpenAIのAIアプリです。 まだ入れていない人は、ここから始めてください。

01

アプリをダウンロード

公式ページからCodexアプリを入手します。MacとWindowsに対応。Macはお使いのチップ （Apple Silicon / Intel）に合うものを選びます。

公式ダウンロードページを開く →
02

サインイン

インストールしたら、お使いのChatGPTアカウントでサインインします。 Codexは ChatGPT Plus / Pro などの有料プランに含まれています。

03

起動を確認

アプリが立ち上がって、メッセージを打てる画面が出ればOK。 ここまで来たら、次のスキル導入に進みます。

How to use

スキルの入れ方と使い方

  1. スキルをダウンロード
    右のボタンから note-banner-maker-skill.zip を保存します。解凍はしなくてOK。
  2. CodexにZIPをアップロード
    Codexを開いて、ダウンロードした ZIP ファイルをそのまま添付（アップロード）します。
  3. 下の指示文をコピーして送信
    この文をそのまま貼り付けて送るだけ。あとはCodexがスキルをセットアップして、 「どんなバナーにする？」「何枚作る？」とヒアリングを始めてくれます。
