Step 0

まずはCodexアプリを入れよう

「Codex（コーデックス）」は、あなたのパソコンの中で作業してくれるOpenAIのAIアプリです。 まだ入れていない人は、ここから始めてください。

01

アプリをダウンロード

公式ページからCodexアプリを入手します。MacとWindowsに対応。Macはお使いのチップ （Apple Silicon / Intel）に合うものを選びます。

公式ダウンロードページを開く →
02

サインイン

インストールしたら、お使いのChatGPTアカウントでサインインします。 Codexは ChatGPT Plus / Pro などの有料プランに含まれています。

03

起動を確認

アプリが立ち上がって、メッセージを打てる画面が出ればOK。 ここまで来たら、次のスキル導入に進みます。

How to use

スキルの入れ方と使い方

  1. スキルをダウンロード
    右のボタンから ZIP を保存し、解凍します。中に note-banner-maker というフォルダが入っています。
  2. Codexのスキル置き場に入れる
    note-banner-maker フォルダを、下の場所にそのまま入れます。

    Mac：Finderで ⌘+⇧+G を押して ~/.codex/skills を開き、その中へ。

    Windows：エクスプローラーで %USERPROFILE%\.codex\skills を開き、その中へ。

    入れたあとの形 → ~/.codex/skills/note-banner-maker/SKILL.md

  3. Codexを再起動
    一度Codexを閉じて開き直すと、スキルが読み込まれます。
  4. 頼むだけ
    Codexに「noteのプロフィールバナーを作って」と送ります。 足りない情報があればCodexの方から質問してくれるので、わかる範囲で答えればOK。 「おまかせで」と言えば全部おまかせでも作れます。

ダウンロード

スキル本体・文字合成スクリプト・仕様メモをまとめた ZIP です。

note-banner-maker-skill.zip

解凍 → フォルダごとスキル置き場へ。

Output

こんなバナーができます

重要な文字や顔・モチーフは、noteで実際に表示される中央の帯にきちんと収まる設計。 設定したときに見切れません。
