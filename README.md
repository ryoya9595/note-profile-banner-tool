# note Banner Studio

noteのクリエイターページ用プロフィールヘッダーを、見切れにくい構図で設計する静的Webツールです。

## できること

- アカウント名、肩書き、ターゲット、強み、価値を入力
- noteヘッダーの安全領域つきライブプレビュー
- AI画像生成用プロンプトの作成
- ランダム案の反映
- SVG形式で保存

## 仕様メモ

- 推奨サイズ: 1280 × 670 px
- アスペクト比: 1.91:1
- 重要な文字や顔は中央表示域に収める前提
- 外部API通信なし
- 入力内容の外部送信なし

## 公開方法

GitHub Pagesでは、リポジトリの `main` ブランチを公開元に設定すればそのまま動きます。
