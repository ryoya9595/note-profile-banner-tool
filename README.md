# noteバナー生成 Codexスキル 簡易版

noteプロフィールヘッダーを `1280×670px` で作るためのCodexスキル配布ページです。

## 内容

- GitHub Pages用の解説ページ
- CodexスキルZIP
- 初心者向けの使い方指示書
- サンプル完成画像

## 方針

- 公式推奨サイズ: `1280×670px`
- 実表示を想定した中央安全帯: `1280×216px`
- 重要な文字・人物・モチーフは中央216px帯に収める
- ガイド版ではなく、完成PNGだけを出す
- 画像生成でズレたサイズはCodex側で最終整形する

## 公開方法

GitHub Pagesでは、リポジトリの `main` ブランチを公開元に設定すればそのまま動きます。
